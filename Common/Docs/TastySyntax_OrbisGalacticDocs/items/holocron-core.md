# Holocron Core

<img width="71" height="73" alt="Holocron Core" src="https://github.com/user-attachments/assets/dbc93618-4896-414a-a1de-4237e271cfa7" />

The Holocron Core is used to craft a [Force Holocron](../the-force/force-holocron.md).

## NPC Trading

To obtain the Holocron Core, visit [Jabba](../vendors/jabba.md) in [Mos Eisley](../planets/mos-eisley.md) and purchase it for `13` Gold Galactic Credits.
