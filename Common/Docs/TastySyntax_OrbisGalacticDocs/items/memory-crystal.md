# Memory Crystal

<img width="73" height="72" alt="Memory Crystal" src="https://github.com/user-attachments/assets/31d47ce9-3f45-40b2-a7e4-a15cafe0bc05" />

The Memory Crystal is used to craft a [Force Holocron](../the-force/force-holocron.md).

## Crafting

It can be crafted at the `Arcanist Workbench` using the following ingredients:

- 50 Essence of Fire
- 50 Essence of Ice
- 50 Essence of Life
- 50 Essence of the Void
- 50 Essence of Water
