# Kyber Crystals

The mod currently adds all of the main colors of Kyber Crystals present in the Star Wars movies and series. All colors can be acquired by mining [Kyber Crystal Ore](../ores/kyber-crystal-ore.md) — the **red** and **white** Kyber Crystals being the exception, which are only available through [Kyber Crystal Bleeding](kyber-crystal-bleeding.md).
