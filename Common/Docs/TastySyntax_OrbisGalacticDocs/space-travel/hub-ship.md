# Hub Ship

The Hub Ship is **GLOBAL** for every server and is used by all players at the same time. Anything you build is persistent.

When you first enter the ship you will be located in the **Hangar**:

<img width="700" height="300" alt="Hytale2026-04-11_13-38-50" src="https://github.com/user-attachments/assets/3d752025-00ca-49a7-ac6c-c15b5e80f911" />

From here, interacting with the HWK-290 again will take you to whichever planet you're currently orbiting. Entering for the first time you will be orbiting **Orbis** (the overworld).
