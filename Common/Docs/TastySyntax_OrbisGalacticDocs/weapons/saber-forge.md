# Saber Forge

[Lightsabers](lightsaber.md) are assembled in the Saber Forge with 2 types of components: a [Lightsaber Hilt](lightsaber-hilts.md) and a [Kyber Crystal](../items/kyber-crystals.md).

<img width="500" height="400" alt="Saber Forge" src="https://github.com/user-attachments/assets/5fb4f76d-da20-4d7c-82b3-79dae53c0a86" />

Inside the Saber Forge you can choose a hilt to assemble your sword with in the first tab:

<img width="700" height="400" alt="Saber Forge Hilt Tab" src="https://github.com/user-attachments/assets/9fbb28ef-9880-4af9-810c-9a9ac2cfbc0f" />

The second tab allows you to pick a color, which then requires the matching Kyber Crystal:

<img width="700" height="400" alt="Saber Forge Color Tab" src="https://github.com/user-attachments/assets/986e03ac-43c7-46dd-a095-c273d3fc770d" />

The Maul hilt has a stance switch setting which allows you to make it a dual-bladed saber instead of a singular one:

<img width="700" height="400" alt="Saber Forge Maul Stance" src="https://github.com/user-attachments/assets/b5e48e1e-82ea-4848-ad83-351ab4c5a392" />
